0. Compelie Vim with pyhon,lua,ruby support
1. Setup [spf-13](https://github.com/rxy0424/spf13-vim)
2. git clone https://github.com/rxy0424/vimrc.git
3. run ./install.py in vimrc directory

5. Complie YouCompleteMe.
6. Have fun
